# Craigslist-React-Project


This is my React project which involves rebuilding Craigslist :)

It's still a work in progress, so I have not yet deployed this project however you're free to take a look in the meantime.

**Steps**
---------

**Download or Pull This Repo**
	Top of this page you can see where it says clone or download

 **Install Node**
	https://nodejs.org/en/

**Download Atom**
	https://atom.io/

 **Install all the node packages** 
On the root of this project run on your terminal (if you want you can do this with yarn but thats optional)
    
    npm install
    
 **Update the node packages** 
On the root of this project run on your terminal (if you want you can do this with yarn but thats optional)
    
    npm update


**Start the server**

  npm run watch
